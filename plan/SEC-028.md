# SEC-028: Predictable Tempfile Name for authorized_keys

## Description
When writing `authorized_keys`, the code uses a predictable tempfile name `authorized_keys.tmp`. If an attacker has write access to the `.ssh` directory, they could create a symlink at this location pointing to a sensitive file, causing data corruption.

## Code Evidence
```cpp
// ssh_manager.cpp:112-113 (authorize_key)
fs::path tmp_path = auth_keys;
tmp_path += ".tmp";  // Predictable: authorized_keys.tmp

// ssh_manager.cpp:180-181 (authorize_public_key_string)
fs::path tmp_path = auth_keys;
tmp_path += ".tmp";  // Same predictable name
```

## Attack Conditions (All Must Be True)
1. Attacker has write permission in target user's `.ssh` directory
2. Attacker creates symlink: `authorized_keys.tmp -> /etc/passwd` (or other sensitive file)
3. Program runs `authorize_key()` or `authorize_public_key_string()`
4. Write operation corrupts the symlink target file

## Mitigating Factors
The `.ssh` directory is set to mode 700 by `ensure_ssh_dir()`:
```cpp
// ssh_manager.cpp:62
auto r2 = utils::exec_safe({"chmod", "700", ssh_dir.string()});
```

This means only the target user (ai-user) has write access. Attackers would need to compromise the ai-user first.

## Remaining Risk
If `.ssh` directory already exists before `ensure_ssh_dir()` is called (e.g., pre-existing directory with wrong permissions), the symlink attack is possible.

## Suggested Fix
Use `mkstemp()` or a random suffix:
```cpp
std::string random_suffix = std::to_string(std::random_device{}());
fs::path tmp_path = auth_keys;
tmp_path += ".tmp." + random_suffix;
```

Or use `open(O_EXCL | O_CREAT)` to fail if file exists:
```cpp
int fd = open(tmp_path.c_str(), O_WRONLY | O_CREAT | O_EXCL, 0600);
if (fd < 0) {
    if (errno == EEXIST) {
        // File exists - potential symlink attack, abort
        return false;
    }
    ...
}
```

## Testing Requirements
- Test: Pre-create `.ssh` directory with mode 777, verify tempfile handling
- Test: Create symlink at tempfile location, verify operation fails safely
